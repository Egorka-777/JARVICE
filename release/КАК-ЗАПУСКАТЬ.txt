J.A.R.V.I.C.E — запуск без npm и без Node.js

1. САМЫЙ ПРОСТОЙ СПОСОБ
   Дважды щёлкните: Запуск-JARVICE.bat
   (откроется окно приложения)

2. ИЛИ напрямую
   Дважды щёлкните: JARVICE.exe

3. УСТАНОВКА В МЕНЮ «ПУСК» (один раз)
   Запустите: JARVICE-Setup.exe
   После установки ярлык появится в меню Пуск как J.A.R.V.I.C.E.

   Если раньше ставили старую версию — удалите через «Программы и компоненты»,
   затем установите JARVICE-Setup.exe заново.

4. АВТОМАТИЗАЦИЯ (кнопка «Запустить»)
   Node.js и npm install больше НЕ нужны.
   Внутри установщика уже есть Node, Playwright и Chromium.

   Первый тест:
   - «Загрузить тест Bing»
   - «Запустить»
   - «Продолжить» в модальном окне
   - «Лог» → Status = done в CSV
